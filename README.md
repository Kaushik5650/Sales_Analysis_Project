## Project WorkflowðŸ’ª

### **Step 1: Data Loading & Cleaning (Python)**
- Load the dataset from `train.csv.zip` using `pandas`.
- Check for missing or invalid data.
- Convert date columns (`Order Date`, `Ship Date`) to proper datetime format.
- Remove duplicates or irrelevant columns.
- Save the cleaned data to **SQLite database (`sales_db.sqlite`)** for SQL analysis.

### **Step 2: SQL Analysis**
- Using **SQLite** via `SQLAlchemy` and `pandas.read_sql()`, we execute queries like:
  - Total sales by **Category** and **Sub-Category**
  - Total sales by **Region** and **Customer Segment**
  - Monthly sales trends
  - Top-selling products

> This demonstrates SQL usage without deep complexity â€” ideal for portfolio projects.

### **Step 3: Python Visualizations**
- Generate visualizations using `matplotlib` and `seaborn`:
  - **Monthly Sales Trend** (line chart)
  - **Sales by Category & Sub-Category** (bar/column charts)
  - **Top Products** (bar chart)
  - **Region-wise Sales** (bar chart)
  - **Segment-wise Sales** (bar chart)
- These charts provide **quick insights** directly from Python.

### **Step 4: Power BI Dashboard**
- Load cleaned CSV or SQLite database into Power BI.
- Create interactive visuals:
  - **Line chart:** Monthly sales trend
  - **Column charts:** Sales by Category and Sub-Category
  - **Bar charts:** Region-wise and Segment-wise sales
  - **Column chart:** Top products
  - **Slicers:** Filter data by Region, Segment, or Category
- Format visuals with titles, labels, and colors for clarity.
- Arrange a **dashboard layout** for professional presentation.

> Power BI adds an **interactive layer** to showcase your insights effectively.
